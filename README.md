# 🍰 Paraíso Gourmet - Cardápio Digital

Este é o cardápio digital sofisticado da **Paraíso Gourmet**, desenvolvido para proporcionar uma experiência visual elegante e luxuosa aos clientes, refletindo a qualidade artesanal e gourmet dos bolos.

## 🎨 Identidade Visual

O design foi construído com foco em sofisticação e leveza, utilizando a seguinte paleta e tipografia:

### Cores Principais
- **Bege Creme/Off-White (`#FDFBF7`)**: Fundo para trazer leveza e limpeza.
- **Marrom Chocolate (`#4B3621`)**: Títulos e detalhes principais, remetendo ao produto.
- **Dourado (`#D4AF37`)**: Molduras e detalhes de destaque para evocar luxo.
- **Verde Sálvia (`#B2AC88`)**: Toques modernos e frescos em seções específicas.

### Tipografia
- **Títulos**: `Dancing Script` (Google Fonts) - Estilo cursivo elegante e artesanal.
- **Corpo e Preços**: `Montserrat` (Google Fonts) - Fonte sans-serif limpa para facilitar a leitura rápida.

## ✨ Funcionalidades

- **Cabeçalho Fixo (Sticky)**: Logo textual e botão de navegação rápida que acompanham a rolagem da página.
- **Seções Categorizadas**: Divisão clara entre as linhas *Clássica*, *Especial* e *Premium*.
- **Personalização Interativa**: Área dedicada para escolha de massas e adicionais gourmet.
- **Call to Action (CTA)**: Botão de destaque para pedidos diretos via WhatsApp.
- **Responsividade Total**: Layout adaptável para smartphones, tablets e computadores.

## 📁 Estrutura de Arquivos

```text
.
├── cardapio.html  # Estrutura semântica e conteúdo do menu
└── style.css      # Estilização, variáveis de cores e animações
```

## 🚀 Como visualizar

1. Clone ou baixe os arquivos do projeto.
2. Certifique-se de ter a conexão com a internet para carregar as fontes do Google e os ícones do FontAwesome.
3. Abra o arquivo `cardapio.html` em qualquer navegador moderno (Chrome, Edge, Safari ou Firefox).
